## Understanding SwiftFusePHP Architecture

SwiftFusePHP follows the MVC (Model-View-Controller) pattern with automatic URL routing:

```
URL Pattern: /Controller/Method/Parameters

Example: /Contact/view/5
├── Controller: Contact
├── Method: view
└── Parameters: [5]
```

The `Core.php` class handles routing automatically:

```php Libreria/Core.php (URL Routing)
class Core
{
    protected $controladorActual = "Inicio";  // Default controller
    protected $metodoActual = "cargaVista";   // Default method
    protected $vista = "index";                // Default view
    protected $parametros = [];
    
    public function __construct()
    {
        $url = $this->getUrl();
        
        // [0] -> Controller
        if(is_array($url) && file_exists("Controlador/" . ucwords($url[0] . "_Controller.php"))) {
            $this->controladorActual = ucwords($url[0]);
            unset($url[0]);
        }
        
        // Require controller file
        require_once "Controlador/" . $this->controladorActual . "_Controller.php";
        $this->controladorActual .= "Control";
        $this->controladorActual = new $this->controladorActual;
        
        // [1] -> Method or View
        if(isset($url[1])) {
            if (method_exists($this->controladorActual, $url[1])) {
                $this->metodoActual = $url[1];
            } else {
                $this->vista = $url[1];
            }
            unset($url[1]);
        }
        
        // [2+] -> Parameters
        $this->parametros = $url ? array_values($url) : [];
        
        // Execute controller method
        call_user_func_array(
            [$this->controladorActual, $this->metodoActual], 
            [$this->vista, $this->parametros]
        );
    }
}
```

## Step 1: Create the Database

First, create a contacts table in your database:

```sql
CREATE TABLE contacts (
    id INT AUTO_INCREMENT PRIMARY KEY,
    name VARCHAR(100) NOT NULL,
    email VARCHAR(100) NOT NULL,
    phone VARCHAR(20),
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4;

-- Insert sample data
INSERT INTO contacts (name, email, phone) VALUES
('John Doe', 'john@example.com', '555-0100'),
('Jane Smith', 'jane@example.com', '555-0101'),
('Bob Johnson', 'bob@example.com', '555-0102');
```

## Step 2: Create the Entity Class

Entity classes represent database records as objects. Create `Modelo/Entidad/Contact_Entity.php`:

```php Modelo/Entidad/Contact_Entity.php
<?php

/**
 * Contact Entity
 * Represents a contact record from the database
 */
class Contact
{
    public $id;
    public $name;
    public $email;
    public $phone;
    public $created_at;
    
    /**
     * Create a Contact object from an associative array
     */
    public static function fromArray($data)
    {
        $contact = new self();
        $contact->id = $data['id'] ?? null;
        $contact->name = $data['name'] ?? '';
        $contact->email = $data['email'] ?? '';
        $contact->phone = $data['phone'] ?? '';
        $contact->created_at = $data['created_at'] ?? null;
        return $contact;
    }
}
```

## Step 3: Create the Model

Models handle database operations. Create `Modelo/Contact_Model.php`:

```php Modelo/Contact_Model.php
<?php
require_once RUTA_APP . "Modelo/Model_Abstract.php";
require_once RUTA_APP . "Modelo/Entidad/Contact_Entity.php";

/**
 * Contact Model
 * Handles database operations for contacts
 */
class ContactMode extends Model
{
    public function __construct()
    {
        parent::__construct();
        $this->entidad = new Contact();
    }
    
    /**
     * Get all contacts
     * @return array Array of contact records
     */
    public function getAll()
    {
        $data = array();
        try {
            // Prepare query
            $this->Conexion->query("SELECT * FROM contacts ORDER BY name ASC");
            
            // Execute and fetch results
            $data = $this->Conexion->getsArray();
            
            if (count($data) == 0) {
                $this->mensaje = "No contacts found";
            }
        } catch (PDOException $ex) {
            $this->mensaje = $ex->getMessage();
        }
        return $data;
    }
    
    /**
     * Get a single contact by ID
     * @param int $id Contact ID
     * @return object|null Contact object or null if not found
     */
    public function getById($id)
    {
        try {
            $this->Conexion->query("SELECT * FROM contacts WHERE id = :id");
            $this->Conexion->bind(':id', $id);
            
            $data = $this->Conexion->getArray();
            
            if ($data) {
                return Contact::fromArray($data);
            } else {
                $this->mensaje = "Contact not found";
                return null;
            }
        } catch (PDOException $ex) {
            $this->mensaje = $ex->getMessage();
            return null;
        }
    }
    
    /**
     * Create a new contact
     * @param array $data Contact data (name, email, phone)
     * @return bool Success status
     */
    public function create($data)
    {
        try {
            $this->Conexion->query(
                "INSERT INTO contacts (name, email, phone) VALUES (:name, :email, :phone)"
            );
            
            $this->Conexion->bind(':name', $data['name']);
            $this->Conexion->bind(':email', $data['email']);
            $this->Conexion->bind(':phone', $data['phone']);
            
            if ($this->Conexion->execute()) {
                $this->mensaje = "Contact created successfully";
                return true;
            } else {
                $this->mensaje = "Failed to create contact";
                return false;
            }
        } catch (PDOException $ex) {
            $this->mensaje = $ex->getMessage();
            return false;
        }
    }
}
```

<Info>
The `Conexion` class automatically detects parameter types in the `bind()` method. It handles integers, booleans, nulls, arrays (as JSON), and strings.
</Info>

## Step 4: Create the Controller

Controllers coordinate between models and views. Create `Controlador/Contact_Controller.php`:

```php Controlador/Contact_Controller.php
<?php

/**
 * Contact Controller
 * Manages contact-related pages and actions
 */
class ContactControl extends Controlador
{
    protected $folder = "contact";
    
    /**
     * List all contacts (default view)
     */
    public function cargaVista($archivo, $data)
    {
        if (count($data) == 0) {
            // Load the list view by default
            $this->lista($archivo, $data);
        } else {
            error(404);
        }
    }
    
    /**
     * Display list of all contacts
     */
    public function lista($archivo, $data)
    {
        $contactModel = $this->modelo('Contact');
        $contacts = $contactModel->getAll();
        
        // Pass contacts to the view
        $parametro = [
            'contacts' => $contacts,
            'mensaje' => $contactModel->getMensaje()
        ];
        
        $this->vista($this->folder . '/lista', $parametro);
    }
    
    /**
     * Display a single contact
     * URL: /Contact/view/5
     */
    public function view($archivo, $data)
    {
        if (count($data) > 0 && is_numeric($data[0])) {
            $contactModel = $this->modelo('Contact');
            $contact = $contactModel->getById($data[0]);
            
            if ($contact) {
                $parametro = ['contact' => $contact];
                $this->vista($this->folder . '/view', $parametro);
            } else {
                error(404);
            }
        } else {
            error(404);
        }
    }
    
    /**
     * Show create contact form
     */
    public function create($archivo, $data)
    {
        $this->vista($this->folder . '/create', []);
    }
    
    /**
     * Handle form submission (AJAX)
     */
    public function save($archivo, $data)
    {
        if ($_SERVER['REQUEST_METHOD'] === 'POST') {
            $contactData = [
                'name' => filterINPUT($_POST['name'] ?? ''),
                'email' => filterINPUT($_POST['email'] ?? ''),
                'phone' => filterINPUT($_POST['phone'] ?? '')
            ];
            
            $contactModel = $this->modelo('Contact');
            $success = $contactModel->create($contactData);
            
            $this->retorno = [
                'mensaje' => $contactModel->getMensaje(),
                'exito' => $success,
                'data' => null
            ];
            
            $this->retornar();
        } else {
            error(404);
        }
    }
}
```

<Note>
The base `Controlador` class provides useful methods:
- `modelo($name)` - Load and instantiate a model
- `vista($path, $params)` - Render a view file
- `retornar()` - Return JSON response for AJAX requests
</Note>

## Understanding the Data Flow

Here's how a request flows through SwiftFusePHP:

<Steps>
  <Step title="User visits URL">
    User navigates to `/Contact/view/5`
  </Step>
  
  <Step title=".htaccess rewrites URL">
    Apache rewrites to `index.php?url=Contact/view/5`
  </Step>
  
  <Step title="Core.php parses URL">
    - Controller: `Contact`
    - Method: `view`
    - Parameters: `[5]`
  </Step>
  
  <Step title="Controller loads">
    `Controlador/Contact_Controller.php` is required and instantiated as `ContactControl`
  </Step>
  
  <Step title="Method executes">
    `ContactControl::view()` is called with parameters `[5]`
  </Step>
  
  <Step title="Model queries database">
    Controller calls `Contact_Model::getById(5)` which uses `Conexion` class to fetch data
  </Step>
  
  <Step title="View renders">
    Controller calls `vista()` which includes `Vista/contact/view_View.php` with contact data
  </Step>
</Steps>

## Advanced Features

### Using Transactions

The `Conexion` class supports database transactions:

```php
public function updateContactWithLog($id, $data)
{
    try {
        $this->Conexion->transaction();
        
        // Update contact
        $this->Conexion->query(
            "UPDATE contacts SET name = :name, email = :email WHERE id = :id"
        );
        $this->Conexion->bind(':name', $data['name']);
        $this->Conexion->bind(':email', $data['email']);
        $this->Conexion->bind(':id', $id);
        $this->Conexion->execute();
        
        // Log the change
        $this->Conexion->query(
            "INSERT INTO contact_logs (contact_id, action) VALUES (:id, 'updated')"
        );
        $this->Conexion->bind(':id', $id);
        $this->Conexion->execute();
        
        // Commit both operations
        $this->Conexion->commit();
        return true;
        
    } catch (PDOException $ex) {
        $this->Conexion->rollBack();
        $this->mensaje = $ex->getMessage();
        return false;
    }
}
```

### Stored Procedures

Call MySQL stored procedures using the `query()` method:

```php
public function getActiveContacts()
{
    $this->Conexion->query("CALL PR_getActiveContacts()");
    return $this->Conexion->getsArray();
}
```

### Input Sanitization

SwiftFusePHP includes the `filterINPUT()` helper function:

```php
$name = filterINPUT($_POST['name']);
// - Trims whitespace
// - Converts 'null' strings to NULL
// - Blocks HTML tags (returns 400 error if detected)
```

<Warning>
Always use `filterINPUT()` on user input and `htmlspecialchars()` in views to prevent XSS attacks.
</Warning>

## Troubleshooting

### Controller Not Found

If you get a 404 error when accessing `/Contact`:
1. Verify file exists: `Controlador/Contact_Controller.php`
2. Check class name is `ContactControl` (Controller name + "Control")
3. Ensure class extends `Controlador`

### View Not Found

If view doesn't load:
1. Check file path: `Vista/{folder}/{view}_View.php`
2. Verify `$folder` property in controller matches directory name
3. SwiftFusePHP looks for `.html` first, then `.php`

### Database Errors

Enable error display temporarily to debug:
```php index.php
error_reporting(E_ALL);
// Comment out: error_reporting(0);
```

Check the error log at `Errores/log.php` for detailed PDO exceptions.
