  <h2 className="text-2xl font-semibold text-gray-900 dark:text-white mb-4">
    Quick Start
  </h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">
    Get SwiftFusePHP up and running in minutes
  </p>

  <Steps>
    <Step title="Download SwiftFusePHP">
      Clone the repository or download the latest release from GitHub:
      
      ```bash
      git clone https://github.com/Cesar-1e/SwiftFusePHP.git
      cd SwiftFusePHP
      ```
    </Step>

    <Step title="Configure your environment">
      Set up your database connection and basic configuration:
      
      ```php Config/configurar.user.php
      <?php
      // Database configuration
      define("HOST", "localhost");
      define("USERNAME", "your_username");
      define("PASSWORD", "your_password");
      define("DATABASE", "your_database");
      define("SSL_CA", false);
      
      // Local directory (if in subdirectory)
      define("LOCALDIR", "");
      
      // SSL configuration
      is_ssl(null); // Auto-detect or set true/false
      ?>
      ```
      
      <Note>
        Update `.htaccess` with your `RewriteBase` path if installing in a subdirectory.
      </Note>
    </Step>

    <Step title="Create your first controller">
      SwiftFusePHP uses an MVC architecture. Create a controller in the `Controlador/` directory:
      
      ```php Controlador/Welcome_Controller.php
      <?php
      class WelcomeControl extends Controlador
      {
          protected $folder = "welcome";
          
          public function index($vista, $params) {
              $data = ["message" => "Hello from SwiftFusePHP!"];
              $this->vista($this->folder . '/index', $data);
          }
      }
      ?>
      ```
    </Step>

    <Step title="Access your application">
      Navigate to your application in the browser:
      
      ```
      http://localhost/SwiftFusePHP/
      ```
      
      The URL routing system will automatically map requests to controllers. Access your Welcome controller at:
      
      ```
      http://localhost/SwiftFusePHP/Welcome
      ```
    </Step>
  </Steps>
</div>
