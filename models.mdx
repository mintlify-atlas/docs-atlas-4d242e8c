## Overview

Models in SwiftFusePHP handle all database operations and business logic. They extend the abstract `Model` class and use PDO for secure database interactions.

## Architecture

SwiftFusePHP's model layer consists of two main classes:

1. **Model** (Modelo/Model_Abstract.php) - Abstract base class for all models
2. **Conexion** (Modelo/Conexion_Model.php) - PDO database connection wrapper

## The Model Abstract Class

<CodeGroup>
```php Model_Abstract.php (Lines 8-27)
abstract class Model{
    protected $Conexion;
    protected $mensaje = null;
    protected $entidad;

    public function __construct(){
        $this->Conexion = new Conexion;
        if($this->Conexion->getError() != null){
            handler(E_ERROR, $this->Conexion->getError(), __FILE__, __LINE__);
        }
    }

    public function getMensaje(){
        return $this->mensaje;
    }

    public function getEntidad(){
        return $this->entidad;
    }
}
```
</CodeGroup>

### Properties

<ParamField path="$Conexion" type="Conexion">
  Database connection instance with PDO wrapper methods
</ParamField>

<ParamField path="$mensaje" type="string|null">
  Stores error or status messages
</ParamField>

<ParamField path="$entidad" type="object|null">
  Optional entity object for data mapping
</ParamField>

## The Conexion Class

The `Conexion` class (Modelo/Conexion_Model.php:5) wraps PDO and provides convenient methods for database operations.

### Connection Setup

<CodeGroup>
```php Conexion_Model.php (Lines 22-40)
public function __construct()
{
    // Configurar conexión
    $dsn = "mysql:host={$this->host};dbname={$this->database};charset=utf8mb4";
    $opciones = [
        PDO::ATTR_PERSISTENT => true,
        PDO::ATTR_ERRMODE => PDO::ERRMODE_EXCEPTION
    ];
    if ($this->ssl_ca) {
        $opciones[PDO::MYSQL_ATTR_SSL_CA] = $this->ssl_ca;
    }

    // Crear una instancia de PDO
    try {
        $this->dbh = new PDO($dsn, $this->usuario, $this->password, $opciones);
    } catch (PDOException $ex) {
        $this->error = $ex->getMessage();
    }
}
```
</CodeGroup>

**Features:**
- Uses MySQL with UTF-8 encoding
- Persistent connections for better performance
- Exception mode for error handling
- Optional SSL support

### Configuration

The connection requires these constants (typically in a config file):

```php
define('HOST', 'localhost');
define('USERNAME', 'db_user');
define('PASSWORD', 'db_password');
define('DATABASE', 'db_name');
define('SSL_CA', null); // Or path to SSL certificate
```

## Database Operations

The `Conexion` class provides methods for all common database operations:

### Preparing Queries

<CodeGroup>
```php Conexion_Model.php (Lines 58-62)
public function query($sql)
{
    $this->stmt = $this->dbh->prepare($sql);
    $this->isExecuted = false;
}
```
</CodeGroup>

### Binding Parameters

<CodeGroup>
```php Conexion_Model.php (Lines 72-95)
public function bind($parametro, $valor, $tipo = null)
{
    if (is_null($tipo)) {
        switch (true) {
            case is_int($valor):
                $tipo = PDO::PARAM_INT;
                break;
            case is_bool($valor):
                $tipo = PDO::PARAM_BOOL;
                break;
            case is_null($valor):
                $tipo = PDO::PARAM_NULL;
                break;
            case is_array($valor):
                $tipo = PDO::PARAM_STR;
                $valor = json_encode($valor, JSON_UNESCAPED_UNICODE);
                break;
            default:
                $tipo = PDO::PARAM_STR;
                break;
        }
    }
    $this->stmt->bindValue($parametro, $valor, $tipo);
}
```
</CodeGroup>

<Tip>
The `bind()` method automatically detects parameter types. Arrays are automatically JSON-encoded.
</Tip>

### Executing Queries

<CodeGroup>
```php Conexion_Model.php (Lines 102-114)
public function execute()
{
    try {
        return ($this->isExecuted ?: $this->isExecuted = $this->stmt->execute());
    } catch (PDOException $ex) {
        $this->error = $ex->getMessage();
        if($this->isTransactionActive()){
            throw $ex;
        }else{
             return false;
        }
    }
}
```
</CodeGroup>

## Transactions

The `Conexion` class provides full transaction support:

<CodeGroup>
```php Conexion_Model.php (Lines 205-251)
// Start transaction
public function transaction()
{
    $this->dbh->beginTransaction();
}

// Execute within transaction
public function executeTransaction($sql)
{
    try {
        $this->query($sql);
        return $this->execute();
    } catch (PDOException $ex) {
        $this->error = $ex->getMessage();
        return false;
    }
}

// Commit transaction
public function commit()
{
    try {
        $this->dbh->commit();
        return true;
    } catch (PDOException $ex) {
        $this->error = $ex->getMessage();
        return false;
    }
}

// Rollback transaction
public function rollBack()
{
    $this->dbh->rollBack();
}

// Check if transaction is active
public function isTransactionActive()
{
    return $this->dbh->inTransaction();
}
```
</CodeGroup>

### Transaction Example

<CodeGroup>
```php Using Transactions
public function transferFunds($fromAccount, $toAccount, $amount)
{
    try {
        $this->Conexion->transaction();
        
        // Deduct from source
        $this->Conexion->query(
            "UPDATE accounts SET balance = balance - :amount WHERE id = :id"
        );
        $this->Conexion->bind(':amount', $amount);
        $this->Conexion->bind(':id', $fromAccount);
        $this->Conexion->execute();
        
        // Add to destination
        $this->Conexion->query(
            "UPDATE accounts SET balance = balance + :amount WHERE id = :id"
        );
        $this->Conexion->bind(':amount', $amount);
        $this->Conexion->bind(':id', $toAccount);
        $this->Conexion->execute();
        
        // Commit if all successful
        if ($this->Conexion->commit()) {
            return true;
        } else {
            $this->Conexion->rollBack();
            $this->mensaje = "Transaction failed";
            return false;
        }
    } catch (PDOException $ex) {
        $this->Conexion->rollBack();
        $this->mensaje = $ex->getMessage();
        return false;
    }
}
```
</CodeGroup>

<Warning>
**Always wrap transactions in try-catch blocks** and rollback on errors to maintain database consistency.
</Warning>

## Error Handling

Models should handle errors gracefully using the `$mensaje` property:

<CodeGroup>
```php Error Handling Pattern
public function create($data)
{
    try {
        $this->Conexion->query("INSERT INTO ...");
        // Bind parameters...
        
        if ($this->Conexion->execute()) {
            return $this->Conexion->dbh->lastInsertId();
        } else {
            $this->mensaje = "Insert failed";
            return false;
        }
    } catch (PDOException $ex) {
        $this->mensaje = $ex->getMessage();
        return false;
    }
}
```
</CodeGroup>

<CodeGroup>

```php In Controller
}
```
</CodeGroup>

## Best Practices

<Tip>
**Always Use Prepared Statements**

Never concatenate user input into SQL queries:

```php
// GOOD
$this->Conexion->query("SELECT * FROM users WHERE id = :id");
$this->Conexion->bind(':id', $id);

// BAD - SQL Injection Risk!
$this->Conexion->query("SELECT * FROM users WHERE id = $id");
```
</Tip>

<Tip>
**Set Error Messages**

Always populate `$this->mensaje` when operations fail:

```php
if (!$success) {
    $this->mensaje = "Clear error message";
    return false;
}
```
</Tip>

<Tip>
**Use Try-Catch Blocks**

Wrap database operations in try-catch to handle PDO exceptions:

```php
try {
    // Database operations
} catch (PDOException $ex) {
    $this->mensaje = $ex->getMessage();
    return false;
}
```
</Tip>

<Warning>
**Avoid N+1 Query Problems**

Use JOINs instead of multiple queries:

```php
// GOOD - One query
SELECT users.*, roles.name as role_name 
FROM users 
JOIN roles ON users.role_id = roles.id

// BAD - N+1 queries
SELECT * FROM users;
// Then for each user:
SELECT * FROM roles WHERE id = ?
```
</Warning>

<Tip>
**Keep Business Logic in Models**

Models should contain business logic, not just database queries:

```php
public function canDeleteUser($userId)
{
    // Check if user has dependencies
    $this->Conexion->query(
        "SELECT COUNT(*) as count FROM orders WHERE user_id = :id"
    );
    $this->Conexion->bind(':id', $userId);
    $result = $this->Conexion->getObject();
    
    return $result->count == 0;
}
```
</Tip>

## Complete Model Example

<CodeGroup>
```php Complete User_Model.php
<?php
require_once RUTA_APP . "Modelo/Model_Abstract.php";

class UserMode extends Model
{
    public function __construct()
    {
        parent::__construct();
    }
    
    public function getAll()
    {
        try {
            $this->Conexion->query("SELECT * FROM users ORDER BY created_at DESC");
            return $this->Conexion->getsArray();
        } catch (PDOException $ex) {
            $this->mensaje = $ex->getMessage();
            return [];
        }
    }
    
    public function getById($id)
    {
        try {
            $this->Conexion->query("SELECT * FROM users WHERE id = :id");
            $this->Conexion->bind(':id', $id);
            return $this->Conexion->getObject();
        } catch (PDOException $ex) {
            $this->mensaje = $ex->getMessage();
            return null;
        }
    }
    
    public function create($name, $email, $password)
    {
        try {
            $hashedPassword = password_hash($password, PASSWORD_DEFAULT);
            
            $this->Conexion->query(
                "INSERT INTO users (name, email, password) 
                 VALUES (:name, :email, :password)"
            );
            
            $this->Conexion->bind(':name', $name);
            $this->Conexion->bind(':email', $email);
            $this->Conexion->bind(':password', $hashedPassword);
            
            if ($this->Conexion->execute()) {
                return $this->Conexion->dbh->lastInsertId();
            }
            
            $this->mensaje = "Failed to create user";
            return false;
        } catch (PDOException $ex) {
            $this->mensaje = $ex->getMessage();
            return false;
        }
    }
    
    public function update($id, $name, $email)
    {
        try {
            $this->Conexion->query(
                "UPDATE users SET name = :name, email = :email WHERE id = :id"
            );
            
            $this->Conexion->bind(':id', $id);
            $this->Conexion->bind(':name', $name);
            $this->Conexion->bind(':email', $email);
            
            return $this->Conexion->execute();
        } catch (PDOException $ex) {
            $this->mensaje = $ex->getMessage();
            return false;
        }
    }
    
    public function delete($id)
    {
        try {
            $this->Conexion->query("DELETE FROM users WHERE id = :id");
            $this->Conexion->bind(':id', $id);
            
            if ($this->Conexion->execute()) {
                return $this->Conexion->getRecord() > 0;
            }
            
            $this->mensaje = "User not found";
            return false;
        } catch (PDOException $ex) {
            $this->mensaje = $ex->getMessage();
            return false;
        }
    }
    
    public function authenticate($email, $password)
    {
        try {
            $this->Conexion->query("SELECT * FROM users WHERE email = :email");
            $this->Conexion->bind(':email', $email);
            $user = $this->Conexion->getObject();
            
            if ($user && password_verify($password, $user->password)) {
                return $user;
            }
            
            $this->mensaje = "Invalid credentials";
            return false;
        } catch (PDOException $ex) {
            $this->mensaje = $ex->getMessage();
            return false;
        }
    }
}
```
